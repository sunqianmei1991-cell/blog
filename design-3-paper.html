class Agent:
    """ReAct Agent"""
    
    def __init__(self, llm):
        self.llm = llm
        
    def run(self, query):
        # 思考-行动-观察循环
        while not self.done:
            thought = self.think()
            action = self.act()
            observe = self.observe()

设计要点

ReAct 循环的核心在于让 LLM 显式地输出思考过程，这对于产品设计的可解释性至关重要。
